# Notes moving pawn

- Pawn #1
	- moving cube with `SetActorLocation`
	- camera: first person
- Pawn #2
	- moving cube with a movement component (`UPawnMovementComponent`?)
	- camera: third person
- Pawn #3

	- camera: third person
- Pawn #4
	- inherit from `DefaultPawn`
	- camera: first person
